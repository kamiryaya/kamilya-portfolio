SQL: клиенты без заказов

Запрос с LEFT JOIN + фильтрация NULL, чтобы найти клиентов без покупок.

SELECT u.id
FROM users u
LEFT JOIN orders o ON o.user_id = u.id
WHERE o.id IS NULL;
SQL

SQL: выручка по статусам

Группировка и суммирование выручки по статусам заказов.

SELECT status, SUM(amount) AS total_amount
FROM orders
GROUP BY status;
SQL
